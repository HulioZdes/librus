# Book Collection Manager

A lightweight Python application for managing a personal book collection.

## Overview

Book Collection Manager is a console application that stores books, displays a collection, calculates reading statistics, and searches for books by title or author.

The project demonstrates object-oriented programming, modular architecture, lists, classes, and formatted console output.

## Features

- Store books
- Search by title
- Search by author
- Reading statistics
- Average rating
- Collection overview
